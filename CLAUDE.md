## What This Project Is

Vectra Icons is a collection of customizable SVG icons for LLM/AI applications. Icons are designed on a 32x32 grid with a 2px stroke width, available in two styles:
- `Icons/Fill/` — filled variants
- `Icons/Stroke/` — outlined variants

Each directory has an auto-generated `iconlist.json` listing all icons by filename.

## Commands

```bash
# Regenerate iconlist.json for both styles
python generate_iconlist_json.py

# Generate icons-metadata.json (requires ANTHROPIC_API_KEY)
ANTHROPIC_API_KEY=sk-... node generate_metadata.js
```

## Architecture

- **SVG icons** live in `Icons/Fill/` and `Icons/Stroke/` (313 icons each)
- **`generate_iconlist_json.py`** scans both icon directories and writes `iconlist.json` files (array of `{ "name": "filename.svg" }` objects)
- **`generate_metadata.js`** uses the Anthropic Batches API to generate `icons-metadata.json` (category, description, tags, aliases for each icon). Kept out of git — run locally as needed.
- **GitHub Actions** (`.github/workflows/update-iconlist-json.yml`) auto-runs the Python script and commits updated `iconlist.json` on any push to `main` that touches `Icons/`

## Adding or Updating Icons

1. Add/modify SVG files in `Icons/Fill/` and/or `Icons/Stroke/`
2. The `iconlist.json` files are auto-generated by CI — you can also run `python generate_iconlist_json.py` locally to regenerate them

## SVG Conventions

- Dimensions: `width="32" height="32"` viewBox
- Stroke icons: 2px stroke width, no fill
- Fill icons: solid black (`#000`) fill paths
